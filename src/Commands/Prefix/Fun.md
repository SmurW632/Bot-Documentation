# Фан команды

### **!картинки** - выполняет поиск изображений в Google по заданному запросу
### *Пример:* 
- ### !картинки [запрос]
- ### !картинки плюшевые мишки - *отобразит коллекцию из 10 изображений по текущему запросу*
